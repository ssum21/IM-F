# 대학 입학전형 교과이수 권장과목 가이드

## 📌 문서 개요

이 문서는 AI가 학생에게 고교학점제 과목 추천을 할 때 참조해야 하는 **대학별 입학전형 교과이수 권장과목** 정보입니다.

### 데이터 출처
- 서울대학교 2026, 2028학년도 전공 연계 과목 선택 안내
- 경희대/고려대/성균관대/연세대/중앙대 공동연구 (2022)
- 건국대학교, 동국대학교, 경희대학교 개별 가이드

### 업데이트: 2026-01-28

## 🎯 핵심 원칙

### 1. 용어 정의
- **핵심 권장과목**: 해당 학과에서 수학하기 위해 **필수적으로** 이수를 권장하는 과목
- **권장과목**: 해당 학과에서 수학하기 위해 **가급적** 이수를 권장하는 과목

### 2. 평가 반영
- 권장과목은 **지원자격이 아님** (미이수해도 지원 가능)
- **학생부종합전형 서류평가**에 반영
- **정시모집 교과역량평가**에 반영
- 이수 여부뿐 아니라 **학습의 깊이와 충실도** 중시

### 3. 학교 미개설 과목
- 학교가 개설하지 않은 과목은 **불이익 없음**
- 단, **공동교육과정** 등 추가 학습 노력은 긍정 평가
- 개설했는데 안 들은 경우와 미개설인 경우를 **다르게 평가**

## 📚 서울대학교

### 2026학년도

#### 인문계열 특징
- 대부분 학과: 핵심권장과목 및 권장과목 **없음**
- 경제학부만 예외: 미적분, 확률과 통계 권장

#### 자연계열 공통 특징
```
핵심: 미적분 (거의 모든 학과)
권장: 확률과 통계 (대부분 학과)
추가: 기하 (공과대학, 자연과학대학 주요 학과)
```

#### 전공별 특이사항

**1. 수학/통계 전공**
- 핵심: 미적분, 확률과 통계, 기하
- 특징: 수학 3과목 모두 핵심

**2. 물리/기계/전기전자/항공우주**
- 핵심: 물리학Ⅱ, 미적분, 기하
- 권장: 확률과 통계
- 특징: 물리학Ⅱ 필수

**3. 화학/화공/재료**
- 핵심: 화학Ⅱ, 미적분
- 권장: 확률과 통계, 기하
- 특징: 화학Ⅱ 필수

**4. 생명과학/의예과/약학**
- 핵심: 생명과학Ⅰ, 생명과학Ⅱ
- 권장: 미적분, 화학Ⅱ, 확률과 통계
- 의예과: 생명과학Ⅰ 핵심, 생명과학Ⅱ/미적분/확률과 통계/기하 권장

**5. 컴퓨터공학**
- 핵심: 미적분, 확률과 통계
- 특징: 물리/화학 과목 없음

**6. 간호대학**
- 핵심: 없음
- 권장: 생명과학Ⅰ, 생명과학Ⅱ

### 2028학년도 (2022 개정 교육과정)

#### 유형 분류

## ⚠️ AI 추천시 주의사항

### 1. 학생 상황 고려
```python
if 학교에서_미개설:
    "학교에서 개설하지 않은 과목은 불이익이 없습니다."
    "가능하다면 공동교육과정을 통해 이수하는 것을 권장합니다."
else:
    "해당 과목은 희망 전공에 중요한 과목입니다."
    "가급적 이수하시는 것을 추천드립니다."
```

### 2. 과목 위계 준수
- 수학: 수학 → 수학I/II → 미적분I → 미적분II
- 과학: 통합과학 → 일반선택 → 진로선택
- **하위 과목 없이 상위 과목만 이수하는 것은 지양**

### 3. 균형잡힌 추천
```
❌ 나쁜 예: "수능 과목만 들으세요"
✅ 좋은 예: "희망 전공 특성을 고려하여 다양한 과목을 이수하세요"
```

### 4. 핵심과목 우선
- 핵심 권장과목 > 권장과목 순으로 우선순위
- 핵심과목이 많으면 학년별로 분산 이수 권장

### 5. 전공 불확실시
```
if 전공_미정:
    "폭넓은 과목 이수를 권장합니다."
    "수학I, 수학II, 미적분I은 대부분 전공에 필요합니다."
    "물리, 화학, 생명과학 중 2개 이상 이수를 권장합니다."
```

## 📖 용어 설명

### 학생부종합전형
- 학생부(생활기록부)를 종합적으로 평가하는 전형
- 교과 성적, 비교과 활동, 자기소개서, 추천서 등 종합 평가
- 권장과목 이수 여부가 **진로역량, 학업역량** 평가에 반영

### 서류평가
- 학생부종합전형의 1단계 평가
- 학업역량, 진로역량, 공동체역량 등을 종합 평가
- 이수 과목의 **위계, 깊이, 관련성** 중점 평가

### 교과역량평가 (정시)
- 정시모집에서 교과 성적을 평가
- 과목 이수 여부 및 성취도 반영
- 서울대는 정시에서도 권장과목 이수 여부 반영

### 공동교육과정
- 여러 학교가 연합하여 개설하는 교육과정
- 소규모 학교에서 개설하기 어려운 과목 제공
- 온라인/오프라인 형태로 운영
